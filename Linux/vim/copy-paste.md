# Copy - Paste

| Standard | Vim |
| --- | --- |
| Cut | Delete |
| Copy | Yank |
| Paste | Put |

* `p` paste after the cursor
* `P` paste before the cursor
* `ddp` swap two lines
* `xp` swap two characters
* `y` yank (copy)
* `yw` yank a word
* `yy` yank an entire line
* `u` undo
* `ctrl r` redo
